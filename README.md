# V2EXIOSAPP
---
##项目介绍

基于V2EX的API，使用 MVVM + ReactiveCocoa 的开源项目，功能还在陆续开发中。

##运行

环境

* xcode 8.3
* cocoapods 1.2
* Swift 3.0
* ReactiveCocoa 5.0.2

项目使用了cocoapods，如果没有安装的话，请自行安装。
clone之后执行

`
pod install                     
`

就可以愉快的玩耍了
